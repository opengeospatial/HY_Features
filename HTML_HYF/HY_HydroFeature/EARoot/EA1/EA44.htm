feature that realises the logical outfall within a hydrologic complex, particularly as a permanent, stable reference location fixed and referenced by coordinates. any geometric object, incl. a point may represent the reference location.<br /></p><p><br /></p><p>Note: the outfall realisation concept implies a hydrologic complex as a collection of related hydrologic features closed under the 'catchment' determination in such that if a outfall realisation exists, these features are in the same hydrologic complex as the outfall they realise. If the realised outfall is topologically connected (i.e. if its corresponding catchment is explicitly not null) then the hydrologic features in the outfall realisation are also connected.<br /></p>
